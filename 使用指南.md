# Material Review - 最终版应用使用说明 (原生兼容版)

## ⚠️ 兼容性修复说明

由于您的 macOS 系统版本较新（macOS 26/16），且为 Apple Silicon (M2/M3) 芯片，之前版本的应用在运行时可能因 Rosetta 转译问题或系统库不兼容而崩溃。

**我们为您重新构建了【全兼容原生版】应用。**

这个版本**内置了独立的、原生的 ARM64 Python 环境**，不再依赖系统自带的 Python，从而彻底解决了崩溃和版本报错问题。

## 📦 分发文件

**位置**: `dist/MaterialReview_Shell.zip` (约 158M)

请将此文件发送给同事。

## 🚀 使用步骤

### 1. 解压与安装

```bash
unzip MaterialReview_Shell.zip
mv MaterialReview.app /Applications/
```

### 2. 首次启动（重要！）

**第一次双击打开应用时，需要初始化内置环境。**

- 您会看到一个通知："正在初始化内置运行环境(原生ARM64)，请稍候..."
- 全程不需要联网，所有依赖包已内置。
- 这个过程可能需要 **1-2 分钟**。
- 初始化完成后，应用界面会自动弹出。

### 3. 注意事项

- **请勿移动应用内的文件**：应用依赖内部目录结构来定位 Python 环境。
- **权限问题**：请确保将应用放在您有写入权限的目录（如 `/Applications` 或 `桌面`），因为首次运行时需要写入配置文件。

## ❓ 常见问题

### Q: 为什么体积变大了？
A: 因为我们内置了一个完整的、独立的 Python 运行环境（约 50MB），以确保在任何 macOS 版本上都能稳定运行，不再受系统升级影响。

### Q: 提示无法打开？
A: 请在终端执行：
```bash
xattr -cr /Applications/MaterialReview.app
```
